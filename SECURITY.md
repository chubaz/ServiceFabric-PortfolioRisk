# Security Policy

Do not report credentials, private portfolio data, provider data, or sensitive
system details through a public issue.

Use the repository's private GitHub security-advisory mechanism for suspected
security vulnerabilities.

Never commit:

- secrets;
- private keys;
- authentication cookies;
- access tokens;
- personal account data;
- licensed provider extracts.

This Day 0 repository is a local research and engineering environment. It is
not a production financial service, public control plane, or trading system.
