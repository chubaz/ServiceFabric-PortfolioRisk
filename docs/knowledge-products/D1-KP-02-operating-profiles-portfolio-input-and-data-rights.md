# D1-KP-02: Operating Profiles, Portfolio Input, and Data Rights

Status: review requested. Deadlines: draft T1+4h; review T1+6h.

The `research` profile uses reproducible reviewed public or labelled synthetic evidence and excludes personal account data by default. The `personal_portfolio` profile accepts only user-supplied input into private local immutable snapshots for monitoring and review. It has no publication, broker, order, trade, or rebalancing route.

Catalogue entries state profile, visibility, and evidence state separately. Private evidence must be private-local. Provider credentials are opaque local references, never catalogue values. Licensed extracts, personal statements, caches, and local databases remain outside Git. Failed or incomplete data stays missing or quality-flagged rather than becoming zero.

Provider enablement, arbitrary SQL, external querying, and portfolio ingestion are outside this planning lane. This record describes disclosure and review requirements only.
