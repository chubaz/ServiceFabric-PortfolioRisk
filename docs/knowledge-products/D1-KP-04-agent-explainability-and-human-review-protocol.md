# D1-KP-04: Agent Explainability and Human Review Protocol

Status: review requested. Deadlines: draft T1+10h; review T1+12h.

An agent-facing timeline identifies the role, registered capability, supplied evidence, assumptions, warnings, limitations, and output digest. Package, capability, tool, application, agent, finding, and alert identities remain distinct. The canonical ServiceFabric invocation and result contracts remain authoritative.

Draft findings and alerts visibly retain synthetic/private/public state and missingness. Consequential next steps require an explicit human decision record. Approval to review a draft grants no authority to call a provider, connect to a broker, submit an order, trade, rebalance, or publish private or licensed data.

This protocol is presentation and review guidance. It does not implement agents, invoke capabilities, or create an alternate runtime path.
