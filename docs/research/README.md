# Day 1 research catalogue

`catalog.yaml` is a reviewed, deterministic metadata catalogue for the Workbench. It records purpose, owner, review status, methodology, inputs, evidence, assumptions, limitations, artifact links, operating profiles, visibility, and public/private/synthetic evidence state.

The catalogue contains no provider observations, licensed extracts, personal portfolio values, or copied source documents. It performs no calculation, query, fetch, or consequential action. A `reviewed` status means the catalogue record is ready for consumption; it does not validate a future application or analytical result.
