# D1-WAVE-1A — Human-readable Workbench

Objective: turn the raw Day 0 prototype into a readable, profile-aware,
accessible server-rendered Workbench while preserving JSON APIs and canonical
ServiceFabric runtime ownership.

Participants: knowledge, experience, integration. Deliver navigation and
screen contracts, reusable presentation primitives, synthetic/profile/review
disclosures, responsive keyboard-accessible HTML, and catalogue-only research
and notebook views. Exclude Node, remote assets, frontend framework migration,
arbitrary execution, and product risk analytics. Acceptance is the 1A gate in
`config/agent/day1/waves.json`.
