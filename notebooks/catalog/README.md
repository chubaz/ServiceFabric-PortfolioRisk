# Day 1 notebook catalogue

`catalog.yaml` contains metadata only. Entries may reserve a future `.ipynb` path, but every entry explicitly states that it is not run. Execution state is limited to `catalogue_only` or `not_available`.

There is no notebook runner, kernel hook, shell command, Python command, SQL command, or arbitrary execution method in this catalogue or its contracts. Evidence and limitations are mandatory. Private-local entries contain no personal values.
